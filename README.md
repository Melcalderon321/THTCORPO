# THT Corporation - Holding Tecnológico

Sitio web corporativo de THT Corporation, un holding tecnológico líder enfocado en la innovación, el desarrollo de talento humano y la transformación digital global.

## Características
- **Diseño Moderno:** Estética "Netflix-style" con tipografía impactante y efectos de ventana de video.
- **Interactividad:** Animaciones fluidas con GSAP y ScrollTrigger.
- **Ecosistema:** Presentación detallada de las unidades de negocio (Tehuentec, THT Ads, THT Devo, etc.).
- **Responsive:** Adaptado para una visualización óptima en dispositivos móviles y escritorio.

## Tecnologías
- HTML5 / CSS3 (Tailwind CSS)
- JavaScript (ES6+)
- GSAP (GreenSock Animation Platform)
- Lucide Icons (Social links)

## Despliegue en Vercel
Este proyecto está optimizado para ser desplegado en [Vercel](https://vercel.com/). Simplemente conecta tu repositorio de GitHub a Vercel y se desplegará automáticamente.

## Estructura del Proyecto
- `index.html`: Archivo principal del sitio.
- `assets/`: Carpeta con imágenes, logos y recursos visuales.
